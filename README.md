Responsive landing page for restaurant build with HTML CSS & JavaScript and ScrollReveal JS library.

## Features
- [x] Hilight Active Link
- [x] Fully responsive
- [x] Dark Mode
- [x] Animation on Scroll
- [x] Custom Scrollbar
- [x] Show and Hide Scroll to top Page
